# Taleq

A platform that helps young adults who stutter build speaking confidence by practicing real-life scenarios and tracking their progress with voice journaling.

## Features

- Indoor challenge prompts to practice techniques 
- Outdoor challenge prompts to practice speaking fluency in real-world situations.
- Voice-Journaling to 

### Requirements

- mimnimum iOS 17.0+
- Mic access for journaling
